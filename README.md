# Library Navigation System

> 图书馆室内导航系统 -- 基于 Wi-Fi 指纹定位，提供图书馆内实时导航与图书检索功能。

## 项目结构

```
├── android/   Android 客户端 (Java)
├── ios/       iOS 客户端 (Objective-C)
├── web/       Web 管理后台 (ThinkPHP)
├── api/       后端 API 服务 (PhalApi)
└── docs/      项目文档与数据库设计
```
